Bienvenido a la API Sistema Solar

🌌 ¡Bienvenido a la API del Sistema Solar! Este proyecto en .NET C# es tu acceso directo a información de los planetas: su nombre, tipo, número de lunas, e incluso su denominación en latín. Explora el cosmos desde tu navegador. 🚀

🔗 Rutas de la API

  • Todos los planetas:
    /api/v1/planetas/all
    Retorna todos los elementos en formato JSON.
  • Buscar por nombre:
    /api/v1/planeta/Júpiter
    Reemplaza Júpiter por otro planeta.
  • Buscar por nombre en latín:
    /api/v1/planeta/latin/Neptunus
    Reemplaza Neptunus por otro nombre en latín.
  • Buscar por tipo:
    /api/v1/planetas/tipo/terrestre
    Ejemplo: terrestre, gigante-gaseoso, gigante-helado.
  • Buscar por cantidad de lunas:
    /api/v1/planetas/lunas/0
    Busca planetas con una cantidad mayor al valor ingresado.
